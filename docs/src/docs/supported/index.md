# 支持的文件

所有支持的解决方案都可以在 [依赖](../#依赖) 找到。



## HTML

`.html / .htm`

使用 html-minifier-terser 进行最小化压缩和配置。



## CSS

`.css / .html<style> / .htm<style>`

使用 clean-css 进行最小化压缩和配置。



## JavaScript

`.js / .html<script> / .htm<script>`

使用 terser 进行最小化压缩和配置。



## JSON

`.json`

使用 JavaScript 原生方法进行最小化压缩。



## XML

`.xml`

使用 minify-xml 进行最小化压缩和配置。



## SVG

`.svg`

使用 minify-xml 进行最小化压缩和配置。
